# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-XX-XX

### Added
- Real-time IV monitoring with SVI parameterization
- Web-based dashboard for volatility visualization
- Options backtesting framework with PnL decomposition
- Minimum variance delta hedging
- SSR (Skew Stickiness Ratio) calculation
- Backtesting environment builder
- Deribit API wrapper (REST and WebSocket)
- Support for long and short options
- Cubic spline fallback for poor SVI fits
- Comprehensive error handling and logging

### Features
- SVI volatility curve fitting with quality assessment
- Model-free PnL decomposition (Delta, Gamma, Vol Block, Vanna Block)
- Real-time term structure visualization
- Historical data collection and storage
- Delta hedging strategies (Black-Scholes and Minimum Variance)
